Inštalácia


  • 1.) Stiahnúť a nainštalovať node http://nodejs.org/

  • 2.) Stiahnúť a nainštalovať grunt npm install -g grunt

  • 3.) Stiahnúť a nainštalovať grunt cli npm install -g grunt-cli

Spustenie


  • 1.) Otvoriť cmd a prejsť do projektu

  • 2.) Projekt musí obsahovať súbory Gruntfile.js a package.json

  • 3.) Pri prvom stiahnutí projektu pomomocou prikazu npm install doinštalovať potrebné moduly

  • 4.) Spustiť sledovanie príkazom grunt watch
